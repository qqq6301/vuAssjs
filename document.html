A Restful API for accessing a real-world data set

Problem:cannot use few server(not from few)


(1)Read

HTTP-Method:GET


parameters:(1)year (2)month
e.g. year=2012&month="November"
or year=2012& month=November
Note:use year=0 and month=0 to retreive all the data

Response:

if sucessful:return the required data and response code:200
if not sucessful(there is not requested data on server):response code:404

(2)Create

HTTP-Method:POST


parameter in json format:
e.g curl -v -X POST -d '{"month":"November","year":"2012","hci p":"3.2","muicp":"2.222","eicp":"2.4"}' http://localhost/ass3.php

Response:

if sucessful:response code:201
if not sucessful(data corrsepending to month and year that user types already exist):response code:404
if json syntax error:response code:400

(3)Modify

HTTP-Method:PUT


parameter in json format:(same as (2))

Response:

if sucessful:response code:200
if not sucessful(data corrsepending to month and year that user types doesn't exist):response code:404
if json syntax error:response code:400

(4)Delete

HTTP-Method:DELETE


parameter in json format:
parameters:(1)year (2)month
e.g. curl -v -X DELETE -d '{"month":"November","year":"2012"}' http://localhost/ass3.php if json syntax error:response code:400 ;

Response:

if sucessful:response code:200
if not sucessful(data corrsepending to month and year that user types doesn't exist):response code:404
if json syntax error:response code:400
